While active, this WordPress plugin replaces all front-end pages with a small mark of respect, and expression of solidarity with George Floyd and so many others.

== Description ==

== Installation ==
 
1. Upload `the-show-must-be-paused.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. To return your website back to normal, simply deactivate this plugin under the `Plugins` menu in WordPress.
